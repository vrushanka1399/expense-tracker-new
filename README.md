# Expense Tracker (Capstone Project)

## 🚀 Features
- User Authentication (Firebase)
- Add, Edit, Delete Expenses
- Role-Based Access Control (Admin/User)
- Audit Logging System (User activity tracking)
- Export to Excel
- Clean Dashboard UI

## 🛠 Tech Stack
- Next.js
- Firebase (Auth + Firestore)
- CSS
